About Us
This website is designed to be a calorie counter to help you manage your diet.
The register, login and logout pages are all dedicated in allowing users to create an account and logging in/out.
Add food allows users to add food to the database and set its typical values, unit of the typical value, calories, carbs, fat, protein, salt, and sugar.
The search page allows users to seach for a food item.
Update food page allows users to update food and if they created the original listing they can also delete the food from the database.
List food simply lists all the food inside the database.
Api page will be listing all foods stored in the database in JSON format
